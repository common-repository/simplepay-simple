Adds SimplePay as a payment method for your WooCommerce store.

== Description ==

Accept payments using SimplePay without leaving your WooCommerce store.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce -> Settings -> Payment Gateways tab and click on SimplePay just below the tabs.
4. Copy and paste the values from your SimplePay account into the appropriate fields and select which brands you have chosen to accept on your SimplePay account.
